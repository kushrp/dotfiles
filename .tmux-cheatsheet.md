# tmux cheatsheet

How to read this:
  ⌃  = Ctrl          (so ⌃a = hold Ctrl, press a)
  ⌃a = the PREFIX. Press Ctrl+a, RELEASE, then press the next key.
       e.g.  ⌃a c   →  Ctrl+a, let go, then c          (new window)
             ⌃a |   →  Ctrl+a, let go, then |          (split left/right)
             ⌃a C-c →  Ctrl+a, let go, then Ctrl+c     (new session)
  Keys with no ⌃a (in copy mode) are pressed directly, vim-style.

## Windows & sessions
  ⌃a c        new window (current dir)
  ⌃a n / p    next / prev window (repeatable)
  ⌃a Tab      toggle last window
  ⌃a 1..9     jump to window N
  ⌃a , / X    rename window / kill session (confirm)
  ⌃a s        session picker (sesh, fzf)   ⌃a S  plain tree picker
  ⌃a C-Space  flip to previous session (sesh)
  ⌃a C-c      new session       ⌃a d   detach (session keeps running)
  ⌃a < / >    move window left / right

## Splits & panes
  ⌃a |        split left/right (keeps cwd)
  ⌃a -        split top/bottom (keeps cwd)
  ⌃a \        full-height vertical split
  ⌃a _        full-width horizontal split
  ⌃h/j/k/l    move between panes — also crosses into nvim splits
  ⌃a h/j/k/l  move between panes (explicit, when not in nvim)
  ⌃a H/J/K/L  resize pane (repeatable — hold ⌃a, tap)
  ⌃a m        zoom / unzoom current pane
  ⌃a o        toggle last pane     ⌃a q   show pane numbers
  ⌃a x        kill pane (confirm)  ⌃a Space  actions menu

## Copy mode (vi keys) → macOS clipboard
  ⌃a [  or  ⌃a Enter   enter copy mode
  v        start selection     ⌃v  block (rectangle) selection
  y        yank → pbcopy (stay) Y   yank → pbcopy and exit
  / ?      search fwd / back    q   quit copy mode
  O        open selected URL / path in browser
  ⌃a ]     paste

## Scroll / search a pane's output
  ⌃a f     Claude Code: dump transcript + search it (type word, Enter)
  ⌃a /     shell / visible output: copy mode + upward search in one key
  in copy mode:  j/k line · ⌃u/⌃d half-page · ⌃b/⌃f page · gg/G top/bottom
                 n/N repeat search · V grab line · y copy · q quit
  (Claude draws on the alt screen, so plain ⌃a [ can't scroll its history — use ⌃a f)

## Claude Code
  ⌃a a     Claude Code dashboard (all agents)
  ⌃a w     new window running claude     ⌃a v  vsplit running claude
  ⌃a f     scroll / search Claude's transcript (see Scroll section)

## Popups / scratch
  ⌃a g     lazygit popup (current repo)
  ⌃a C-g   scratch shell popup (throwaway zsh, current dir)
  ⌃a t     scratch SESSION popup (persistent "scratch" — notes/repl)
  ⌃a e     extrakto: fuzzy-grab any path / URL / word on screen
  ⌃a C-h   THIS cheatsheet        ⌃a ?  searchable list of ALL binds

## Misc
  ⌃a r     reload ~/.tmux.conf    ⌃a C-a  send a literal Ctrl-a to the shell
  ⌃a I     install plugins        ⌃a U    update plugins

## Sessions persist automatically
  resurrect + continuum auto-save every 15 min and restore on start.
  Manual: ⌃a C-s save · ⌃a C-r restore
